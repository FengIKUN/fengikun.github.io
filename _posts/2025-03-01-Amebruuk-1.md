### 二、特殊规则

- ​长音字母**使用两点符号**​（Ä, Ë, Ï, Ö, Ü），如确实遇到输入环境不便等问题，允许使用双写元音字母来代替，但正式场合仍应使用两点符号。

- ​促音x的二铁律：

  - 仅出现在**辅音**前：Ka**x**te（✓） / **X**ate（✗）

  - **禁止词首**位置：**X**amuk（✗）

- ​颤音R的发音梯度：

  - 词首：**​rr**uam [ˈru.am]（强颤音）

  - 词中：ha**r**ank [ha.ˈraŋk]（中等颤动）

  - 词尾：ama**r**​ [a.ˈmar]（弱颤动）

## 第二章：名词系统

所有的名词基本形式都以"-k"结尾。

### 一、格位体系（先变格后变复数）

| 格位   | 后缀    | 示例（主格→变形）        | 功能            |
|-------|---------|-------------------------|----------------|
| 主格   | ∅     | Mizuk（水）              | 主语/主题       |
| 宾格   | -a     | Mizuka​（水-宾格）        | 直接宾语        |
| 与格   | -aya   | Mizukaya​（给水）         | 间接对象        |
| 属格   | -n     | Mizukn​ [mizu.kən]       | 所属关系        |

### 二、复数系统

| 数量 | 后缀 | 示例（主格→复数） | 逻辑 |
| ---- | ---- | ---------------- | ---- |
| 双数 | -s   | Mizuks（两汪水）   | 原始双数标记 |
| 复数 | -sm  | Mizuksm（多片水域） | 集合标记   |

叠加示例：

- 与格复数 → Mizukaya**sm**​（给人的多片水域）

- 属格双数 → Mizuk**ñas**​（两汪水的所属）

### 三、提示和特殊规则

- 属格变为双数或复数时，为了避免出现"nsm"这样的拼写，**将属格结尾的"-n"改为"-ña"再加后缀**（发音规则不变），如 Harank → Harank**n** → Harank**ñas** [ha.ˈran.kəns] / Harank**ñasm** [ha.ˈran.kən.səm]

- **有生性规则：**

  - 有生名词（人/动物）：**与格必须显式标记**，如 Marukaya（给人）→ **必须用与格**

  - 无生名词：与格可省略，如 Bönk（给书）→ **可直接用宾格**

## 第三章：动词系统

### 一、核心结构

**变位顺序：否定 → 时态 → 体 → 人称**，即**动词原形 + 否定标记 + 时态标记 + 体标记 + 人称词尾**。

例：Kïm（吃）→ Kïm + asen + t + ä + ai → Kïmasentäai（他当时没吃完）

### 二、时态和体的后缀

动词的时态变位如下表。

| 时态   | 后缀  | 示例  | 含义  |
| ---- | ---- | ---- | ---- |
| 现在时 | ∅ | Kïm  | 现在吃 |
| 过去时 | -t | Kïmt | 过去吃 |
| 将来时 | -r | Kïmr | 将要吃 |
| 不定时 | -s | Kïms | 泛指任何“吃”的动作 |

每个时态都对应了一般体、进行体、完成体三种体的变位（不定时没有进行体和完成体）。动词的体变位如下表（以过去时为例）。

| 体     | 后缀  | 示例  | 含义  |
| ---- | ---- | ---- | ---- |
| 一般体 | ∅ | Kïmt  | 过去吃（一般动作） |
| 进行体 | -ï | Kïmtï | 过去正在吃 |
| 完成体 | -ä | Kïmtä | 过去吃完了 |

### 三、人称后缀

| 人称 | 单数 | 复数 | 例词（以现在时为例） |
| ---- | ---- | ---- | ---- |
| 第一 | -i | -ine | Kïmi（我吃），Kïmine（我们吃） |
| 第二 | -u | -une | Kïmu（你吃），Kïmune（你们吃） |
| 第三 | -ai | -ane | Kïmai（他吃），Kïmane（他们吃） |

## 第四章：基础句法

### 一、人称代词与指示代词

Amebrük语中使用如下人称代词。注意**代词也有和名词一样的变位**。

| 人称 | 单数 | 双数 | 复数 |
| ---- | ---- | ---- | ---- |
| 第一 | Maruk | Maruks | Maruksm |
| 第二 | Apruk | Apruks | Apruksm |
| 第三 | Maink | Mainks | Mainksm |

指示代词有三种，分别是 Ruk（近），Rik（较近），Rak（远），可以分别翻译为“这个”，“这个”，“那个”。

### 二、SOV语序与助词

基本结构：**[主格名词] + i + [宾格名词] + [动词]**

**例句**：Maruk i bönka ruami.（我拿书。）

Maruk → 我

i → 表示主语结束的助词

bönka → 名词“书”的宾格

ruami → 动词“拿”的第一人称单数形式（以后缩写为1-sg）

### 三、话题化与强调

我们使用​助词 **zo** 标记一个**话题**：

**例句**：Amebrükn zo, yümai älal.（至于枫叶的，它是美丽的。）

Amebrükn → 名词“枫叶”的属格

zo → 标记话题的助词

yümai → 系动词“是”的3-sg形式

älal → 形容词“美丽”

​宾语前移强调：当强调句中的宾语时，将**主语和宾语倒装**。

Bönka i Maruk ruami → 强调"书"被拿
